1.gitについて

gitは、プログラムのソースコードなどの変更履歴を記録・追跡するための分散型バージョン管理システムです。
バージョン管理システムでは、「誰が・いつ・どのように変更をソースコードに加えたか」という情報を、「バージョン」として記録しておきます。

gitの特徴として挙げられることは、

  1. 分散型バージョン管理システム

    2. リモートサーバーなどにある中心リポジトリの完全なコピーをローカル環境に作り出し、ローカルリポジトリを使って作業できます。
    （リポジトリとは、プロジェクトの保存場所。プロジェクトに必要な複数のファイルやすべての変更履歴を保持することです。）

  2. GitはCUI仕様

    3. CUIとはキーボードで入力するコマンドによって操作するツールのことで、複数人で同じ事象を共有することに役立てられます。

  3. セキュリティが高い

    4. Git は、管理ソースコードの整合性を大事にして作成されており、SHA1と呼ばれる暗号化された安全なハッシュアルゴリズムで守られています。

  4. 柔軟性が高い

    5. 色々な種類の非線形開発ワークフローのサポート、小規模および大規模両方のプロジェクトでの効率性、
    さらに数種類の既存システムおよびプロトコル（手順）との互換性と柔軟な使い方が期待できます。

2.htmlについて

HTMLとは,「ハイパーテキスト・マークアップ・ランゲージ（Hyper Text Markup Language）」のことで、
WEBページを作成するための言語です。
”マークアップ”とは文章が構成されている要素がどのような役割を持っているのかを指示します。
コンピュータに理解できるよう意味を持たせてあげるために記号として

HTMLタグ

が目印をつけるための記号として使用されます。

文字に意味を持たせることができます。
